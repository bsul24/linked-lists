# Linked List

A small JavaScript linked list implementation built for The Odin Project.

## Features

- Append and prepend nodes
- Get size, head, tail, and values by index
- Search for values with `contains()` and `findIndex()`
- Remove nodes with `pop()` and `removeAt()`
- Insert one or more values with `insertAt()`
- Jest tests for the main linked list behavior

## Run Tests

```bash
npm test
```
